Foreach Loops

Loop through arrays using <!--foreach()-->:

Simple Loop (Value Only)

        
            <!--foreach($products as $product)-->
            <div>
            <h3><!--$product[name]--></h3>
            <p><!--$product[price]--></p>
            </div>
            <!--endforeach-->

Loop with Key and Value

        
            <!--foreach($items as $key=>$value)-->
            <p>Item #<!--$key-->: <!--$value[name]--></p>
            <!--endforeach-->
